L'IDE simple et collaboratif

Atom Teletype

De grandes choses se produisent lorsque les développeurs travaillent ensemble; de l'enseignement et du partage des connaissances, à la création de meilleurs logiciels. Le télétype pour Atom rend la collaboration sur le code aussi facile que de coder seul, directement depuis votre éditeur.
Partagez votre espace de travail et modifiez le code ensemble en temps réel. Pour commencer à collaborer, ouvrez Teletype in Atom et installez le logiciel.

Un éditeur open-source

Atom est aussi un éditeur qui avant tout colaboratif en ce sens qu'il est open-source. Vous pouvez, quand vous le souhaitez et à votre échelle, participer au développement de Atom. Vous trouverez plus bas dans la page le lien vers le Github d'Atom avec la possibilité de participer à celui-ci.
Ce partage rejoint vraiment la philosophie d'Atom qui, au delà d'être un outils de productivité, est aussi un outil de création et de conception à part entière.

Une communauté très active

"Atome est flexible et peut être largement hacké et modifié. Le système de packages d'Atom permet de trouver facilement des configurations personnalisées faites par l'utilisateur, et il peut inclure une coloration syntaxique et des suggestions automatiques pour presque toutes les langues."

"Atom est formidable en ce sens qu'il s'agit d'un logiciel open-source, ce qui garantit qu'il peut être personnalisé sans restrictions. Il est intégré à GitHub, ce qui est très pratique pour les développeurs web. Il dispose d'une interface graphique moderne et légère et permet d'installer des thèmes tiers."

"Son principal avantage est que le fait d'être un outil open source le rend totalement gratuit, ce qui permet à d'économiser une grande quantité d'argent. Il garantit un bon codage abrégé. Il offre une excellente interface utilisateur, il est personnalisable."

Le modèle économique

Atom comme vous l'avez compris est un éditeur de code entièrement gratuit et qui fonctionne sous le mode de l'open source. Cet éditeur n'a pas de modèle économique à notre connaissance compte tenus du fait que cet éditeur de texte se base essentiellement sur les apports de sa communauté.

Conclusion

En conclusion, Atom est un éditeur de texte qui est open-source très plébicité par la communauté des développeurs. Cet intérêt se retrouve dans le fait que cet éditeur est open-source mais aussi par le fait qu'il possède des fonctionnalités particulière comme le Teletype.
