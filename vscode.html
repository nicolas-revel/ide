Intelli Sence

Au-delà de la mise en évidence syntaxique et de l'autocomplétion, IntelliSense va plus loin en proposant des complémentions intelligentes basées sur les différents types de variables, les définitions de fonctions et les modules importés.

Deployez votre site en toute confiance

Avec Microsoft Azure, vous pouvez déployer et héberger vos sites, stocker et interroger des données relationnelles et documentaires, et évoluer avec l'informatique sans serveur, le tout en toute simplicité, depuis le VS Code.

Quelques fonctionnalités devenues basiques

Visual Studio Code offre la possibilité de travailler facilement avec git et d'autres fournisseurs de MCS. Visual Studio Code possède ainsi une intégration git qui facilte l'utilisation de github comme beaucoup d'IDE récents.

Déboguer le code directement depuis l'éditeur. Lancez ou attachez à vos applications en cours d'exécution et déboguez avec des points d'arrêt, des piles d'appels et une console interactive.

Customiser son expérience avec le large catalogue d'extensions. Les extensions fonctionnent selon des processus distincts, ce qui garantit qu'elles ne ralentiront pas votre éditeur.

Modèle éconmique

Visual Studio Code est un IDE entièrement gratuit sans offre premium. C'est un grand avantage pour cet IDE par rapport aux autres parce que c'est un éditeur puissant, avec beaucoup de fonctionnalités importantes et pratiques.

Conclusion

En conclusion, Visual Studio Code est un éditeur avec encore de beaux jours devant lui. Il fait partie du haut du panier des éditeurs de code gratuit et présente des fonctionnalités très importantes et pratiques. A voir comment avance le modèle économique à l'avenir.
